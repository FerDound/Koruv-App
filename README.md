Koruv 
Red social escolar desarrollada para fortalecer 
la comunidad estudiantil.

Tecnologías
- Kotlin + Jetpack Compose
- Firebase (Firestore, Auth)
- Cloudinary (almacenamiento de imágenes)
- Arquitectura MVVM

Características
- Feed de publicaciones
- Chat en tiempo real
- Perfiles de usuario
- Sistema de seguidores
- Notificaciones

Desarrolladores
- Castillo Navarro Jose Fernando
- Diaz Hurtado David

Nota
Para usar este proyecto necesitas configurar 
tu propio google-services.json y credenciales 
de Cloudinary.
